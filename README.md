## Usage
```bash
  -faults
        Active/disable faults
  -latency int
        Latency in milliseconds to be added to fault type slowness (default 1)
  -microservice string
        Microservice Endpoint (default "http://www.google.com")
  -percent float
        Percentage of faults
  -type string
        Type of the value (default "error")
```
